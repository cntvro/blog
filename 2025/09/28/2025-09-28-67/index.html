debian12安装后要做的事

发表于2025-09-28|更新于2025-12-04|Debian
|浏览量:
1、VI方向键不能用。

解决办法：卸载VI，重新安装
apt remove vim-common -y
apt install vim -y
2、禁用DVD源
sudo nano /etc/apt/sources.list
3、配置 Sudo 用户
su - root
apt install sudo -y
usermod -aG sudo <用户名>
4、更新系统
sudo apt update && sudo apt upgrade -y
sudo reboot
5、安装sensors 命令行工具：lm-sensors（CPU温度核心工具）
sudo apt-get install lm-sensors -y

scp 传输文件
本地主机和服务器之间需要进行传输文件可以使用 ssh 的方式来进行，通过scp 命令行来进行设备之间的文件传输。 将文件从 src 发送到 dest 的命令如下，需要交互式的输入对应的密码： scp 将 Linux 文件发送到本机上(1)、将单个文件从Linux 服务器传输到本机 scp userName@192.168.0.11:/home/TmpDir/Myfile “D:\TransferData\MyData" (2)、将整个目录传输到本机上，-r 表示递归发送目录中的文件 scp -r userName@192.168.0.11:/home/TmpDir “D:\TransferData\MyData" 将本机的文件发送到Linux服务器上，只需互换 src 和 dest 即可，如：scp D:\123\favicon.ico root@10.10.10.5:/var/www/ 参考链接
