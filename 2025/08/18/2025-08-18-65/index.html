Debian安装sudo

发表于2025-08-18|更新于2025-12-04|Debian
|浏览量:
1.查看是否安装

dpkg -l|grep sudo
2.安装
1.切换到root用户
su - root
2.安装sudo
apt-get install sudo
3.配置
安装好后，配置允许使用sudo的用户
sudo vim /etc/sudoers
找到如下内容
text

User privilege specification

root ALL=(ALL:ALL) ALL
在下面新增一行，允许其他用户使用sudo，fanqi是实际的用户名：
fanqi ALL=(ALL:ALL) ALL
